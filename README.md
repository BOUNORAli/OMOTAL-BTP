# OMOTAL TRAVAUX - Gestion de chantiers

Socle initial de la plateforme web et mobile de gestion de chantiers OMOTAL TRAVAUX.

## Demarrage local

```bash
npm.cmd install
npm.cmd run dev
```

Puis ouvrir `http://localhost:3000`.

## Scripts

```bash
npm.cmd run typecheck
npm.cmd run test
npm.cmd run build
```
